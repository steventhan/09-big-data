Big Data: geoJSON

The goal for this assignment is to simulate a data set that you may encounter during project week!

Let's focus only on transforming this collection. No need to invoke a constructor, or pass off control to Handlebars.

Start simple:

1. Transform the initial collection into and array of each object's properties.

2. Next, try combining that new collection with some other valuable info such as the coordinates for each zip.

3. Also remove redundancies like the county, borough, city and state since we already know the context, as our app would theoretically show, too.

4. For the zips where we have two or more neighborhoods, let's eliminate all but the first one ("Greenwich Village, SoHo" becomes "Greenwich Village").

5. Populate an array of the five 'densest' neighborhoods based on zip occurrences.
